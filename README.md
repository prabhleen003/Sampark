# Sampark(Ongoing)
# open for contributions

A community platform built with React, Express, and MongoDB.

## Project Structure

```
sampark/
├── client/   ← React app (Vite + Tailwind + GSAP)
└── server/   ← Node.js + Express API + MongoDB
```

## Getting Started

### Backend
```bash
cd server
npm install
cp .env.example .env   # fill in your values
npm run dev
```

### Frontend
```bash
cd client
npm install
npm run dev
```

## Environment Variables

See `server/.env.example` for required backend variables.
